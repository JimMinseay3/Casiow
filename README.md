# Casiow 邮件发送系统

## 项目简介

Casiow 邮件发送系统是一个基于 Python 和 Tkinter 开发的图形界面邮件群发工具。该系统支持多账户管理、Excel 表格导入收件人信息、批量发送邮件、附件添加等功能，适用于企业、学校等需要定期发送通知邮件的场景。

## 功能特性

- 多账户管理：支持配置多个邮箱账户，可随时切换
- 图形界面操作：直观易用的 Tkinter 界面
- Excel 导入：支持从 Excel 文件导入收件人列表
- 批量发送：支持大量邮件的自动批量发送
- 附件支持：可添加一个或多个附件到邮件
- 发送记录：记录每次发送的日志信息
- 错误处理：完善的异常处理机制

## 使用方法

### 运行程序

有两种方式可以运行程序：

1. **直接运行可执行文件**：
   - 解压 `Casiow邮件发送系统_包含批处理文件版.zip`
   - 进入解压后的文件夹
   - 双击运行 `Casiow邮件发送系统.exe`

2. **使用批处理文件**：
   - 解压 `Casiow邮件发送系统_包含批处理文件版.zip`
   - 进入解压后的文件夹
   - 双击运行 `启动邮件发送系统.bat`

### 配置邮箱账户

首次使用需要配置邮箱账户信息：
1. 在程序界面点击"账号管理"按钮
2. 添加新的邮箱账户（目前支持 QQ 邮箱和 Outlook 邮箱）
3. 输入邮箱地址和授权码（注意是授权码而非密码）
4. 保存配置后即可使用

## 配置文件说明

系统使用 `email_accounts.json` 文件存储邮箱账户信息。该文件会在首次运行时自动生成，也可手动编辑。

## 系统要求

- Windows 7 或更高版本操作系统
- 网络连接（用于发送邮件）
